## booker

A fast, simple & powerful book framework, powered by Node.js.

基于node.js实现的一个 web book 生成工具。

## install

```node
npm install booker.js -g
```


## usage

* 初始化项目

```bash
booker init
```

* 文件编译

```bash
booker g
```

* 本地查看

```bash
booker server -w
```
